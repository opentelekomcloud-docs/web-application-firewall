Rule Configurations
===================

This section describes how to configure protection rules.

.. note::

   When you are modifying the configuration of a policy bound to two or more domain names, the **Modify Policy** dialog box is displayed, indicating that the modified policy configuration applies to multiple bound domain names. See :ref:`Figure 1 <waf_01_0007__fig10336172412375>`. If you want to modify the policy configuration, click **Yes**. To stop displaying this dialog box again, select **Do not show again**.
